Dive In Time

Dive in Time is an interactive project which turns Time into a Rive and allows the audience to start his/her/its own journey in Time.

- Single click to start the BGM;
- Press Any key to Dive in Time;
- ↑ and ↓ to mive up and down;

- A summery woule be generated and auto-downloaded at the end
- It should be a "txt" file named Dive in Time, the file is SAFE
Enjoy your journey! and Cherish the moment.
